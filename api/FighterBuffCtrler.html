3.3增加 Buff控制器

方法

speedUp

传入参数：upVal:Number = 0, hold:Number = 5

返回类型：void

说明：设置速度增加（参数1：增加倍数，参数2：持续时长）

使用示例：

x
1
parent.$fighter_ctrler.getTargetSP().getCurrentTarget().getBuffCtrl().speedUp(0.3,15);
